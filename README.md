# 🎵 Zara's Music Journey

A gamified multi-instrument practice tracker, calendar-aware scheduler, and self-learning motivator — built by a product mom for her 9-year-old aspiring singer.

## The Problem

My daughter Zara is 9 and learning **four instruments** (vocals, drums, guitar, keyboard). Managing practice across four instruments, four teachers, four class schedules, plus vacation conflicts, doctor appointments, and playdates — while keeping a 9-year-old motivated to practice independently — is a genuine product challenge.

Nothing on the market addressed: multi-instrument scheduling, calendar-aware conflict detection, gamification that motivates self-directed learning, or automatic cancellation/makeup reminders based on each teacher's policies.

## The Solution

### 🏠 Dashboard
Practice cards per instrument with streak tracking, XP system, and combined Musician Level. Daily alerts when calendar conflicts need action.

### 📅 Calendar-Aware Scheduling
Connects to **Mom's Outlook** and **Zara's Gmail** to pull vacations, doctor appointments, and playdates. Auto-detects conflicts with music classes. Shows cancellation deadlines per teacher's policy and makeup windows. Sends reminders to both parent and child.

### 🌳 Skill Trees per Instrument
Sequential milestones for Vocals, Drums, Guitar, and Keyboard — each unlocking XP.

### ⏰ Self-Set Practice Schedule
Zara sets her own goals per instrument per day. Autonomy drives motivation.

### 🎓 Class Policies
Each teacher's class day, time, cancellation policy, and makeup window in one place.

### 🔍 Discovery Mode
Suggests tutorials matched to current skill level. Self-directed learning.

### 🎶 Concert Setlist
Songs she can perform end-to-end. Her dream concert, built one song at a time.

### 🔥 Gamification
XP, Musician Level, streak tracker with confetti (her demand), and Stage Mode at 7-day streaks.

## What I Learned

1. **Gamification only works when the user sets their own goals.**
2. **Calendar integration is a product differentiator, not a feature.**
3. **The best motivation system celebrates consistency, not perfection.**
4. **Your toughest user researcher is a 9-year-old with zero filter.**
